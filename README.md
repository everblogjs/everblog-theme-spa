## everblog-theme-spa

SPA theme for [everblog-adaptor-spa](https://github.com/everblogjs/everblog-adaptor-spa).

### Usage

1. create `_config.yml`

![](./public/images/1.png)

2. create `xxx.md`

![](./public/images/2.png)

3. Start everblog

```sh
DEBUG=* everblog start
```
